# 千依子剑

简称：千依 <br/>
千依子剑**不能简称**「子剑」，因为通常说的「子剑」是指「莫言」（五绝）的「子剑东流」

## 石壁有字

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;《千依子剑》 <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;家学渊源重，女子亦英雄。 <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;千种柔情在，一招破长空。

## 现实世界

认证机构信息管理系统 中的 「历史记录」

## 武侠世界

扶桑女子、服部千依的独门武功

## 作者笔记

```text
不怎么修改的、常被引用的基础表，会运用「千依」这种技术
使用者对数据的可靠性就有了自信，因为如果谁动了就有历史

千依的意思是：千条历史记录，以子表的形式、依主数据而存
```
